FAQ

What is Litecoin?
Litecoin is a peer-to-peer cryptocurrency and open source software project released under the MIT/X11 license. Inspired by and technically nearly identical to Bitcoin (BTC), Litecoin creation and transfer is based on an open source encryption protocol and is not managed by any central authority. Litecoin is intended by its developers to improve upon Bitcoin and offers three key differences.
What is Litecoin mining?
Litecoin mining is the process of using a computer to process transactions for the Litecoin network. By doing this you can attempt to solve a ‘block’ which pays out a reward. Currently the Litecoin reward per block is 50 Litecoins.
Why ASIC is the fastest technology solution for cryptocurrency mining process?
Users have used various types of hardware over time to mine blocks. The possible solutions are:
  • CPU Mining: Early Litecoin client versions allowed users to use their CPUs to mine. The advent of GPU mining made CPU mining financially unwise. The option was therefore removed from the Litecoin client.
  • GPU Mining: GPU Mining is drastically faster and more efficient than CPU mining.
  • FPGA Mining: FPGA mining is a very efficient and fast way to mine, comparable to GPU mining and drastically outperforming CPU mining. FPGAs typically consume very small amounts of power with relatively high hash ratings, making them more viable and efficient than GPU mining.
  • ASIC Mining: An application-specific integrated circuit, or ASIC, is a microchip designed and manufactured for a very specific purpose. For the amount of power they consume, they are vastly faster than all previous technologies
